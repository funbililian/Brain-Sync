# Brain-Sync - QUIZ APPLICATION FRONTEND DEVELOPMENT PROJECT

## Project Overview

The Quiz Application is a web-based platform that allows users to answer multiple-choice questions, receive instant feedback, track their scores, and view quiz results. The application is designed to provide an engaging and user-friendly experience while demonstrating frontend development skills such as responsive design, DOM manipulation, state management, and API integration.

---

## Live Demo

[Live Demo](https://brain-sync-three.vercel.app/)

---

## Project Objectives

### Main Objectives

- Build a responsive and interactive quiz application.
- Provide a seamless user experience across devices.
- Display quiz questions dynamically.
- Calculate and display scores in real time.
- Allow users to review their performance after completing a quiz.

---

## Key Features

### User Features

#### 1. Landing Page

- Welcome message
- Application description
- Start Quiz button

#### 2. Quiz Interface

- Display questions dynamically
- Multiple-choice answer options
- Next and Previous buttons
- Progress indicator

#### 3. Timer Feature

- Countdown timer for each quiz
- Auto-submit when time expires

#### 4. Score Calculation

- Instant score tracking
- Percentage calculation

#### 5. Results Page

- Final score display
- Correct and incorrect answers
- Performance summary

#### 6. Restart Quiz

- Option to retake the quiz
